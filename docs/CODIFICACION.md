# Codificación y nombres técnicos

Todos los documentos, scripts y archivos de texto del repositorio usan
codificación UTF-8 para conservar correctamente acentos, eñes y demás
caracteres del español.

Los nombres técnicos de archivos, carpetas, columnas y variables permanecen en
ASCII cuando forman parte de comandos o rutas, por ejemplo:

```text
metricas_holdout_todos_los_modelos.csv
comparacion_modelos_validacion_cruzada.csv
proyecto-amr-ecoli
```

Esto evita romper scripts, enlaces y comandos entre Windows, Linux, Google
Colab y GitHub. El texto descriptivo y visible para las personas sí utiliza
ortografía española completa.
