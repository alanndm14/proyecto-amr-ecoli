# Paquetes fuente integrados

El repositorio fue ensamblado a partir de estos paquetes entregados:

- `proyecto genomica.zip`: datasets, script optimizado y resultados principales.
- `mlp_amr_resultados_final.zip`: resultados y script original del MLP.
- `robustez_metodo_laboratorio_resultados (1).zip`: resultados leave-method-out.
- `amr_figuras_tablas_finales (1).zip`: figuras y tablas finales.
- `final genomica.pdf`: informe final.

Los contenidos relevantes fueron extraidos y organizados en `data/`, `scripts/`,
`results/`, `figures/`, `report/` y `docs/`. Los ZIP originales permanecen en
la carpeta Descargas del equipo y no se duplican dentro del repositorio.
